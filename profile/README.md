Collection of Libraries that allow for webview usage on the desktop

### Repositories

| Repository                                                    | Description                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------|
| [PhotinoEx](https://github.com/PhotinoEx/PhotinoEx)           | Complete C# library for Webview desktop apps                  |
| [Site](https://github.com/PhotinoEx/PhotinoEx.Site)           | Static site for PhotinoEx                                     |
| [Docs](https://github.com/PhotinoEx/PhotinoEx.Docs)           | Detailed Documentation for using PhotinoEx                    |
| [Samples](https://github.com/PhotinoEx/PhotinoEx.Samples)     | Sample usage for PhotinoEx                                    |
| [Templates](https://github.com/PhotinoEx/PhotinoEx.Templates) | Templates for dotnet new, quick and easy init of new projects |
